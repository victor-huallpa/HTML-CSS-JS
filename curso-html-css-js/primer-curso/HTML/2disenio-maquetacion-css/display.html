Propiedad DISPLAY

    La propiedad display define como se comporta un elemento destro del flujo del documetno. Existen varios valores importantes:

  • block: Ocupa todo el ancho disponible y comienza una nueva linea. ejemplo 'div, p, h1'
  • inline: Ocupa solo el ancho necesario y no fuerza un salto de linea. Ejemplo 'span, a'
  • inline-blcok: se comporta como una linea, pero permite modificar dimensiones(width, height)
  • none: Oculta completamente el elemento, como si no existiera en el DOM
