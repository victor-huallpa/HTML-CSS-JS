Unidades de medida

en CSS existen dos tipos de unidades de media ABSOLUTA y RELATIVA

Unidades ABSOLUTAs

px pixeles, tamanio fijo sin importar el zoon

unidades RELATIVAS

  • % relativo al tamanio del contenedor padre
  • em relativo al tamanio de fuente del elemento padre
  • rem relativo al tamanio de fuente del documento html
  • hv/ vw relativo al alto (hv) o ancho (vw) de la ventana
