POSICIONAMIENTO BASICO

  1. static Posicionamiento predeterminado (flujo normal)
  2. relative Se mueve respecto a su posicionamiento original
  3. absolute Se posiciona respecto a su primer elemento padre con posicion relativa
  4. fixed Se mantiene fijo en la pantalla
  5. sticky Se mantiene fijo al hacer scroll hasta que llegue a un limite

NOTA: recuerda que puedes usar top, right, bottom, left para posicionar un elemento
