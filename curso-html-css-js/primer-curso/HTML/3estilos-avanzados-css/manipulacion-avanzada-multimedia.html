MANIPULACION AVANZADA DE MULTIMEDIA(imagenes y videos)

para entender un poco mejor de que trata este apartado, es la responsividad de los videos y imagenes, la daptabilidad que tieene en diversos dispositivos y para eso se utiliza la propiedad object-fit

objetect-fit

Esta propiedad css contyrola como se ajusta el contenido de un elemento reemplazado(como img y videp) dentro de sus contenedor

    valores de objetect-fit

  • cover: se usa para redimencionar el multimedia al contenedor, y en caso este no se de las dimenciones a escala del conetendor el multimedia se recortara para llenar el contenedor

  • contain: escala la el multimedia sin recortar sus dimenciones, y en caso no coinsida con el contenedor este ulitmo mostrara espacios en blancop>

  • fill: el multimedia escala para llenar todo el contendor sin mantener la escala dimencional que tiene lo que lo puede llevar a distorcionar si no coincide la escala de la imagen y el contenedor

  • scale-down: la iamgen escala de manera dimencional, para entrar en todo el contenedor, esto osurre solo si el contenedor es mas grande que el multimedia caso contrario el mulitmedi mantendra sus dimenciones

  • none: El multimedi no escala y solo se muestra en sus dimenciones originales, si el mulitmedia es mas grande que el contenedor este se recortara al tamanio del contenedor
