PROPIEDADES DE COLOR Y FONDO

Como se lee en el titulo de la pagina a los elementos se les puede dar color y fondo dcon las siguente definicion de colores

  1. NOMBRE: red, blue, black, etc
  2. RGB: rgb(255, 0, 0)
  3. HEXADECIMAL: #ff0000
  4. HSL: hsl(0, 100%, 50%)

ejemplo

nombre: {color: red;}
rgb: {color: rgb(0, 4, 255);}
hexadecimal: {color: #ffffff;}
hsl: {color: hsl(138, 100%, 50%);}
