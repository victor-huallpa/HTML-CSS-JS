FLEXBOX

flexbox es un sistema de disenio que permite distribuir elementos de manera eficiente dentro de un contenedor flexible. Ideal para centrar elementos, alinear itemsy distribuir espacio.

Propiededades claves de flexbox

    Se aplica al contenedore padre (display: flex)

  • justify-content: Alineacion horizontal (start, end, center, space-between, space-around)
  • align-items: Alineacion vertical (start, end, center, stretch)
  • flex-wrap: Define si los items deben ajustarse o quedarse en una sola linea

    Se aplica a los elementos hijos (flex-item)

  • flex-grow: Permite a un ítem crecer si hay espacio disponible.
  • flex-shrink: Define si un ítem puede reducirse si es necesario.
  • flex-basis: Tamaño base del ítem antes de aplicar grow o shrink.
