GRID LAYOUT

Css grid permite crear disenios de cuadricula avanzados con un control preciso sobre filas y columnas.

    Propeidades principales

  1. display: grid: Convierte un contenedor en una cuadrícula.
  2. grid-template-columns: Define el número y tamaño de columnas.
  3. grid-template-rows: Define el número y tamaño de filas.
  4. gap: Espacio entre filas y columnas.
  5. grid-column y grid-row: Controlan la posición de los ítems en la cuadrícula.
