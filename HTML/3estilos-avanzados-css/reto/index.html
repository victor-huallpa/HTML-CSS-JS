Reto Bloque 3

Este párrafo se redefine mediante CSS avanzado, donde la tipografía personalizada, cargada con `@font-face`, establece un tono único. Sombras y degradados sutiles, aplicados con `text-shadow` y `linear-gradient`, resaltan palabras clave. Animaciones de desvanecimiento y desplazamiento, creadas con `keyframes`, se activan al pasar el cursor, suavizadas por transiciones `transition`. La función `calc()` ajusta el espaciado según el tamaño de la pantalla, mientras que `clip-path` recorta el texto en formas inusuales. Variables CSS mantienen la coherencia, y filtros aplicados mediante transiciones, generan efectos visuales al pasar el cursor.
