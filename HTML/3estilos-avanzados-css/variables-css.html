USO DE VARIABLES EN CSS

Las CSS permiten definir valores reutilizables para colores, tamanios y otros estilos

Esto se define en la hoja desetilos usando la siguiente sintaxis: :root{--color-1: #ff0000;--color-2: #00ff00;--color-3: #0000ff;}
